# ticketpark

__팀명 |__ ticketpark

__팀원 이름 |__ 김민선, 김소정, 한동민, 조하란, 오준표

__프로젝트 기간 |__ 2021. 12. 13.~ 2022. 01. 07.

__프로젝트 소개 |__ 

📌 TicketPark는 콘서트, 뮤지컬, 연극 등 공연에 대한 클라이언트의 예매에 대한 결제를 진행합니다. 
    또한 공연과 관련된 문화생활을 영상, 칼럼과 같은 매체를 통해 최신 소식을 소개합니다.
